Background:

The Sporting Goods Sales Data for Q1 2023 presents sales figures for the leading five brands, along with quarterly growth rates. This comprehensive dataset encompasses the companys nationwide presence and is organized by region and city, offering a detailed insight into its performance during this period.

Objective:

Perform PCA on Q1 2023 sales data for the top 5 brands, categorized by Region and City_Code, aiming to reduce dimensionality while retaining essential information and uncovering latent patterns and relationships among variables.

Summarizing Performance By Region

# Group the data by region and calculate the average performance for each region
summary_pc_region<-data %>%
  group_by(Region) %>%
  summarize(performance = round(mean(performance),3))%>%
  as.data.frame()

# Create a bar graph using the specified color palette
ggplot(summary_pc_region, aes(x = Region, y = performance, fill = Region)) +
  geom_bar(stat = "identity") +
  labs(title = "Average Performance by Region", x = "Region", y = "Average Performance") +
  theme_minimal() +
  theme(axis.text.x = element_text(angle = 0, hjust = 1)) +
  scale_fill_brewer(palette = "Set2")

Observation:

The analysis of the data reveals notable regional disparities in performance metrics, with the WEST region exhibiting the most pronounced positive value, signifying exceptional performance, whereas the EAST region records the lowest negative value, indicating subpar performance relative to other regions.
