Customer Segmentation¶

Created on 10-08-2022¶

Background: The transaction data for channel partners(customer) of a large FMCG company is available. The data available is for two years -by month of the year and brand. The marketing team wants to create segments of customers based on transaction pattern so that marketing strategies can be developed for each segment.¶

Objectives:¶

  1. To create segments of similar customers which can be used for marketing strategies

Data Description:¶

Column Name Description
Customer ID Customer ID for each customer
Month Month of transaction
Brand Brand purchased
Year Year of transaction
Sales Total sales in INR

Libraries used:¶

Library Application
pandas To handle data frames in python
matplotlib/seaborn Plotting graphs
numpy Adding support for large, multi-dimensional arrays
sklearn.preprocessing Used for transforming/standardizing data
sklearn.cluster To use KMeans algorithm
sklearn.decomposition To run Principal Component Analysis (PCA)
warnings To disable warnings

Create plot for elbow method to decide optimum number of clusters¶

In [16]:
distortions = []
K = range(2,10)
for k in K:
    kmeanModel = KMeans(n_clusters=k)
    kmeanModel.fit(df_scaled)
    distortions.append(kmeanModel.inertia_)
    
plt.figure(figsize=(16,8))
plt.plot(K, distortions, 'bx-')
plt.xlabel('k')
plt.ylabel('Distortion')
plt.title('The Elbow Method showing the optimal k')
plt.show()

Comment : Using elbow method, K=4 clusters are finalised¶
