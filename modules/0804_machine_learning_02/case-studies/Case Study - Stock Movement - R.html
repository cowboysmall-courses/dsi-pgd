Predictive Model for Stock Movement - Case Study

Background: This case study is a continuation of ML1 case study. Decision Tree, Support Vector Machines and Neural Networks are used to predict the stock direction.

Machine Learning Algorithms are very useful to predict movement of stock-either to predict price or direction. In Spite of a lot of uncertainty, the well trained models can help to develop successful trading strategies. In this case study, we consider one of the leading global IT companies-Infosys and predict its next day movement using various ML algorithms.

Objective: To predict closing price direction of Infosys stock on day t+1 relative to day t .

Model 3. Neural Networks

Building Model

normalize <- function(x) {
  return ((x - min(x)) / (max(x) - min(x)))
}

traindata$Returns<-normalize(traindata$Returns)
traindata$CloseToOpen<-normalize(traindata$CloseToOpen)
traindata$HighToLow<-normalize(traindata$HighToLow)
traindata$Std_deviation<-normalize(traindata$Std_deviation)
traindata$Returns_ndq<-normalize(traindata$Returns_ndq)

library(neuralnet)

nn_bank <- neuralnet(Direction_next ~  Returns + CloseToOpen + HighToLow + 
                    Std_deviation + Returns_ndq,data=traindata,
                     hidden=2,err.fct="ce",linear.output=FALSE)

Plotting neural network
